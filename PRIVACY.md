# Privacy Policy

**Last updated: 2026-06-30**

## Summary

NotePad Web collects no data, makes no network requests, and contains no telemetry.
All files and settings stay in your browser. Nothing is transmitted anywhere.

## Data storage

All data is stored exclusively on your own device using:

- **IndexedDB** — stores open tab state and buffer contents for session restore.
- **chrome.storage.local** — stores user preferences (theme, font size, etc.).
- **File System Access API** — reads and writes files you explicitly choose via
  the file picker. File handles may be retained in IndexedDB only to re-open the
  same file in a future session; this requires your explicit permission grant.

## Network access

NotePad Web declares no host permissions and makes zero outbound network requests.
The extension operates fully offline after installation.

## Third parties

No third-party analytics, crash-reporting, advertising, or tracking SDKs are
included. The Monaco editor (bundled MIT library) operates entirely in-process
with no network calls.

## Chrome Web Store

If you installed NotePad Web from the Chrome Web Store, Google's standard Chrome
Web Store privacy policies apply to the installation process itself. NotePad Web
itself has no relationship with those policies.

## Changes

Any material change to this policy will be reflected in a new version published
to the Chrome Web Store with an updated "Last updated" date above and a
corresponding entry in [CHANGELOG.md](CHANGELOG.md).
